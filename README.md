# react-native-ad-manager

A React Native wrapper around Google Ad Manager SDKs

## Installation

```sh
npm install react-native-ad-manager
```

## Usage

```js
import AdManager from "react-native-ad-manager";

// ...

const result = await AdManager.multiply(3, 7);
```
